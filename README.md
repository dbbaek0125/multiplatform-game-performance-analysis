# Statistical Analysis of Video Game Trends

## Overview
This project analyzes a dataset of video games (2004–2010) to study platform distribution and review score trends using statistical inference.

## Key Questions
- What proportion of video games are multiplatform?
- What is a reasonable range for this proportion?
- Is the typical review score lower than 75?

## Methods
- Proportion estimation
- Confidence interval (simulation-based)
- Hypothesis testing (median using simulation)

## Results
- Approximately 50% of games are multiplatform
- 80% confidence interval: (0.49, 0.52)
- Strong evidence that median review score < 75 (p-value ≈ 0)

## Tools
- Python (pandas, matplotlib)
- Jupyter Notebook
